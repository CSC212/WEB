Part 1: Making our own webpage. AnÂ application programming interface(API) is a set of routines, protocols, and tools for building software applications.

For an API to be created, there should be a model, controller and an interface. A model is where much of the code is done. It communicates with the database and send info to the controller. The controller is the middle man. It interacts with the model and send info from the model to the viewer or interface and vice versa The view/interface gives the output of the code in a user friendly manner. In this project we are using php as a main programming language for the creation of the web service
